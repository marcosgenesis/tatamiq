# Platform admin separate from academy membership

Tatamiq will implement platform administration as a separate area from the academy/instructor experience, using Better Auth's Admin plugin and the global `admin` user role instead of Better Auth organization membership. The first version starts from a configured **Administrador da Plataforma**, and that administrator may promote or remove other platform administrators through the UI, as long as the system never removes the last active administrator. Separate operator/support roles are deferred until there is a real need. Platform administrators are not academy members by default; customer-scoped access must happen through temporary, audited assisted support instead of changing `member.role` or relying on `activeOrganizationId`.
